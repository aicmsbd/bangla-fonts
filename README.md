---

## Quick Start — CDN Usage (jsDelivr)

Copy this `<link>` into your HTML `<head>` — no download needed:

```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/aicmsbd/bangla-fonts@main/bangla-weights.css">
```

Then use in CSS:

```css
body {
  font-family: "Bangla", system-ui, sans-serif;
}
```
